L'application a besoin d'être reliée à ta base de données Supabase (gratuit). Ouvre le fichier assets/config.js et remplace les deux valeurs par celles de ton projet.

  1. Crée un projet sur supabase.com
  2. Lance le script supabase-schema.sql (SQL Editor)
  3. Copie Project URL et la clé anon public dans assets/config.js

Le guide complet est dans le fichier SETUP.md du projet.
